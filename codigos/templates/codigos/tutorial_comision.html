¿Como funcionan las comisiones?

Definamos los siguientes terminios:
Recarga: Compra de nuevos codigos.
Saldo acumulado: Dinero que no alcanzo para la ultima recarga pero se usara para la proxima recarga.
Comision: Porcentaje de ganancia

Al monto de una recarga se le sumara el porcentaje de comisión correspodiente y el saldo acumulado, y de esta nueva cantidad se le daran la cantidad equivalente en códigos segun el precio de venta. En caso de que las recargas no sean exactas al numero de codigos dados, el dinero pendiente se guardará como saldo acumulado para la proxima recarga

Ejemplo

Datos iniciales
Comision: 15%
Saldo acomulado inicial: $0
Precio del codigo: $10
Recarga de $100

Recarga + comisión de recarga + saldo acumulado inicial--> 100.00 + 100.00x0.15 + 0.00 = 115.00

nueva_cantidad = 115.00

nueva_cantidad entre el precio de venta = 115.00 / 10.00 = 11 codigos + 5 pesos de saldo acumulado
